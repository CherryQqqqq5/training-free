# Outputs

Runtime-generated BFCL traces and reports stay under `outputs/`.

- `outputs/bfcl_v4/`: baseline and patch run roots
- `outputs/reports/`: mined failure JSONL and derived reports
- `outputs/artifacts/phase1/`: tracked artifact templates and protocol-facing examples

Most generated files remain git-ignored. Tracked files under `outputs/artifacts/` exist only to document artifact shape.
